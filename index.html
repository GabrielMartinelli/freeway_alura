FREEWAY GRANDPARENTS

Instruções

Freeway Grandparents é uma cópia do jogo eletrônico Freeway do console Atari 2600. Você controla os dois personagens tentando atravessar uma rodovia sem ser atingido pelos automóveis. Se o personagem atravessar a rodovia ele recebe uma vida e se for atingido por um veículo ele perde uma vida.

* O jogo só possui versão desktop

Como jogar

  • Para mover o personagem 1 (Vovô) utilize as teclas W(cima) A(esquerda) D(direita) S(baixo).
  • Para mover o personagem 2 (Vovó) utilize as setas.

Desenvolvedor

O jogo foi desenvolvido por Gabriel Martinelli no curso ministrado pelo professor Guilherme Lima na plataforma Alura.
